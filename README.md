# Projet fil Rouge
## 2025 -2026 


**Objectif du projet :**

Créer une plateforme qui permet à des utilisateurs authentifiés via Google OAuth2 de créer des événements, 
de gérer des listes de participants, d’organiser des sessions ou des ateliers au sein des événements, 
et de suivre les inscriptions ou la participation. Cette plateforme inclut un backend robuste basé sur Spring Boot 3 
et une interface utilisateur interactive construite en Angular.`
